## Vim Config

For plugins. 
Update the yaml and run the command below:

```
python2 setup.py -h                                                   
usage: setup.py [-h] mode pluginsFile

positional arguments:
  mode         init/update
  pluginsFile  path to plugins.yml

optional arguments:
  -h, --help   show this help message and exit
```
